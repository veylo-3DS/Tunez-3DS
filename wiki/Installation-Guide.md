# Installation Guide

1. Download the latest release from the [Releases page](https://github.com/veylo-3DS/Tunez-3DS/releases/latest).
2. For CIA: Scan the QR code with FBI or transfer the file to your SD card and install via Title Manager.
3. For 3DSX: Place the file in your sdmc:/3ds/Tunez3DS/ folder and run it from the Homebrew Launcher.
